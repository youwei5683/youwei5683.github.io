一、安装及配置过程

$ wget https://github.com/SoftEtherVPN/SoftEtherVPN_Stable/releases/download/v4.29-9680-rtm/softether-vpnclient-v4.29-9680-rtm-2019.02.28-linux-x64-64bit.tar.gz 
> tar -zxvf softether-vpnclient-v4.29-9680-rtm-2019.02.28-linux-x64-64bit.tar.gz
> cd vpnclient
$ make    -->所有都选择"1.yes"最后会编译成功
$ mv vpnserver /usr/local
$ cd /usr/local/vpnclient/
$ chmod 600 *
$ chmod 700 vpncmd
$ chmod 700 vpnclient

　　四、验证命令

#ifconfig                                                   ——>查看是否获取到ip
#dhclient vpn_vpn  　　　　　　                               ——>找dhcp获取ip
#ifconfig                                                   ——>现看看是否成功
#sudo ip route add 61.39.160.58 via 192.168.1.1 dev eth0    --->(这里看你使用的是有线还是无线网络) ——>将VPN服务器IP添加到eth0)
#sudo ip route del default                                  ——>删除默认路由
#sudo ip route add default via 192.168.30.1                 ——>添加默认路由 192.168.30.1 为VPN服务器拨通后的网关按需修改
这样就大功告成！将我们的一台机器加入到我们的VPN中了，就可以实现我们想要做的东东了~~~~

五、一些售后服务

1、添加开机启动VPN服务

#vi /etc/init.d/vpnclient   -------->>新建启动脚本
#!/bin/bash
vpnclientpath=/usr/local/vpnclient
if [ ! -w $vpnclientpath ]
then
sudo chmod 777 $vpnclientpath
fi
cd $vpnclientpath
./vpnclient start
exit 0
2、设置VPN启动后自动连接VPN服务 
#./vpncmd                                     ——>选择2进入管理
VPNClient>AccountStartupSet       　　　　　　  ——> 设定连接设置的VPN启动连接
假若出现某些问题，你可以试试看
假若你不可以访问网络，你必须保证你的防火墙（尽量不要关闭，但在测试的时候可以试试关闭看看）可以让vpn通过： 
#./vpnclient stop
