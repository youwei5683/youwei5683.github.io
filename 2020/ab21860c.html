Hexo默认使用的文章永久链接
2020年5月7日 下午
233 字 3 分钟

本文最后更新于：2020年5月8日 上午

在Hexo博客上添加可爱的Live 2D模型

npm install --save hexo-helper-live2d

x.安装 hexo-admin，实现网页发布文章:

npm install --save hexo-admin

然后打开 http://localhost:4000/admin/ 就可以看到管理页面

x. 安装abbrlink插件，在博客根目录（执行hexo命令的地方，解决文件名是中文的问题）安装插件：

npm install hexo-abbrlink --save

修改 _config.yml 添加：

# abbrlink config 让文章链接变成随机名字
url: http://yoursite.com
root: /
permalink: :year/:abbrlink.html
permalink_defaults:
  author_name: index
  
abbrlink:
  alg: crc32  # 算法：crc16(default) and crc32
  rep: hex    # 进制：dec(default) and hex

x .让 hexo在后台运行。安装pm2:

$ npm install -g pm2

写一个执行脚本
在博客根目录下面创建一个hexo_run.js

//run
const { exec } = require('child_process')
exec('hexo server',(error, stdout, stderr) => {
        if(error){
                console.log('exec error: ${error}')
                return
        }
        console.log('stdout: ${stdout}');
        console.log('stderr: ${stderr}');
})

运行脚本 在根目录下执行:

# pm2 start hexo_run.js
